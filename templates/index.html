get /counter?word={word}&url1={url1}&url2={url2}
Returns occurrence of word in one or both urls provided. (counterwordwordurl1url1url2url2Get)

Path parameters

word (required)
Path Parameter — The word to search for occurrences.
url1 (required)
Path Parameter — The first link to search for occurrences.
url2 (required)
Path Parameter — The second link to search for occurrences.

Return type

inline_response_200_1

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Successful response inline_response_200_1

400

The specified word is invalid (e.g. not a word)

404

A page with the specified url was not found
